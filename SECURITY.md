# Security policy

Report suspected credential access, path traversal, unsafe package replacement, hidden camera/photo access, or data-exposure issues privately to the repository owner before public disclosure.

Do not include live credentials, private keys, cookies, tokens, or personal photos in a report. Use synthetic fixtures and redacted diagnostics.

Qpet is a skills-only plugin. It does not operate a developer-owned server, collect telemetry, maintain user accounts, or run a background usage monitor.
