# Qpet support

Before reporting a problem:

1. Update the Qpet marketplace.
2. Reinstall Qpet.
3. Start a new Codex task.
4. Confirm the bundled image-generation and pet-production skills are available.

When opening an issue, include:

- Codex surface and version;
- operating system;
- the user prompt with personal data removed;
- the validation error or failed step;
- whether the input was keywords, a user-attached selfie, or another authorized reference.

Never attach API keys, authentication files, cookies, full logs containing credentials, or an unredacted personal photo.
