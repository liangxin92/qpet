# Support

## Before contacting support

Run the plugin's runtime check and package validator, then keep the JSON result. Do not include a selfie, API key, account token, or unrelated personal file in a support report.

Useful information includes:

- plugin version
- Codex/ChatGPT surface and app version
- operating system
- the exact user-visible error
- runtime-check or validation JSON with personal paths redacted
- whether the issue affects concept generation, image attachment, animation, validation, or installation

## Common fixes

- If the pet workflow is missing, open **Settings → Pets → Create your own pet** once, then retry.
- If the camera action is unavailable, attach an existing photo instead.
- If validation reports the wrong atlas height, do not mix the web v1 export with the desktop v2 package.
- If a local pet id already exists, choose a new id or explicitly confirm replacement after making a backup.

Use the contact form or monitored contact method on the hosted version of this support page. Before marketplace submission, the verified publisher must host this page at a public HTTPS URL and add that URL to the listing.
