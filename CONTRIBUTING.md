# Contributing

Keep changes focused on Qpet's 3D chibi pet workflow.

Before proposing a change:

1. Run the unit tests.
2. Run `python3 scripts/verify_marketplace.py`.
3. Build the release archives.
4. Confirm that no generated personal photos or local pet packages are committed.

Changes that weaken consent, access credentials, scan photo libraries, patch the Codex client, or silently overwrite an existing pet are not accepted.
